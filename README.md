# Zipeador

Welcome to your new gem!


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'zipeador'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install zipeador

## Usage
```ruby
Zipeador.zip(file)
Zipeador.unzip(zipfile)
```
